相关文件：`llvm-project/mlir/include/mlir/IR/BuiltinTypes.td`


主要的内置类型包括：

| 类型 (Type)                 | 语法示例                 | 说明                                                                                                      |
| --------------------------- | ------------------------ | --------------------------------------------------------------------------------------------------------- |
| **整数 (Integer)**    | `i32`,`i8`,`i1`    | 表示任意位宽的整数。`i1`通常用于表示布尔值。                                                            |
| **浮点数 (Float)**    | `f32`,`f64`,`bf16` | 表示标准的 IEEE 浮点数格式，以及 `bfloat16`等。                                                         |
| **None**              | `none`                 | 表示“无类型”，类似于 C++ 中的 `void`。通常用于不返回值的函数。                                        |
| **复数 (Complex)**    | `complex<f32>`         | 表示复数，由两个相同类型的元素（实部和虚部）组成。                                                        |
| **元组 (Tuple)**      | `tuple<i32, f32>`      | 元素的集合，每个元素可以有不同的类型。                                                                    |
| **向量 (Vector)**     | `vector<4xf32>`        | 固定大小的一维数组，用于 SIMD (单指令多数据流) 操作。                                                     |
| **张量 (Tensor)**     | `tensor<4x16xf32>`     | 多维数据数组，是表示深度学习计算的核心类型。可以是固定形状，也可以是动态形状（如 `tensor<?x16xf32>`）。 |
| **内存引用 (MemRef)** | `memref<4x16xf32>`     | 指向一段结构化数据的引用（类似指针），包含了布局（layout）和形状信息。常用于表示内存中的 buffer。         |
| **函数 (Function)**   | `(i32, f32) -> f64`    | 定义了函数的输入参数类型和返回结果类型。                                                                  |
